# 波特价值链

## 一句话总结

9 类活动把公司拆开。任务是：找出竞争优势在哪一环形成——成本、差异化或两者兼有；并指出**联动**在哪里相互加强、在哪里相互打架。

## 填写顺序

1. **进货 → 服务**（从左到右）先填。把运营骨架画上去，再讨论优势。
2. **支持活动**（企业基础设施 → 采购）。它们看似"无聊"，但通常是最难复制的护城河。
3. 回过头**给每个活动标颜色**：黄=成本优势、蓝=差异化、紫=联动、粉=漏点。
4. **利润**区最后填——汇总公司到底靠什么挣钱。

## 跨块不变量

- 联动是真正的优势所在。生产运营 ↔ 采购 之间的一张紫色联动 sticky，比两个填满了"特性"的格子更值钱。
- 支持活动不是可选项。一家 HRM 薄弱的公司，无论营销 sticky 怎么写都做不出服务导向的战略。
- 同一活动同时有成本 + 差异化优势是可能的（如 Patagonia 的可持续采购既是成本纪律也是品牌差异化），但少见——遇到时用双色标记。

## 反模式

- 通用流程图，没有做优势评估。
- 单独评估每个活动，跳过联动颜色。
- 因为"无聊"跳过支持活动。
- 平台 / 服务业务：假装进货和出货物流还重要——它们已经接近零；技术开发 + 容量采购才是主导。
- 分析停在画布上，没有翻译成 BMC 的 KA / KR / KP。

## 语气

活动级具体性。「生产运营」本身不够——写明是**哪种**生产（高混合柔性制造？7×24 自动化云服务？）。数字和动词胜过形容词。

## Blocks

`zoneId` 是 API 的稳定标识符,`pingarden canvas write` payload 必须用它,不要翻译。

### `firm-infrastructure` — 企业基础设施

**Prompt** — 总体管理、财务、法务与治理。如何支撑或限制下方的主要活动？

**Example** — 全球财务与资本配置

**Quality bar** — 总体管理、财务、规划、法务、公司治理，以及全公司依赖的任何集中职能。 问的是：基础设施层在支撑还是限制下方每一个主要活动？ 基础设施日常不可见，却决定了公司能扩到什么规模。 多业务公司的连接组织住在这里。

### `human-resource-management` — 人力资源管理

**Prompt** — 招聘、培训、薪酬、文化与留任。人才管线在哪一环定胜负？

**Example** — 专科招聘 + 学徒管线

**Quality bar** — 招聘、培训、薪酬、文化、留任和领导力发展。 问的是：哪些能力扎根在人身上？ 人才管线在哪个环节决定战略的成败？ HRM 是最容易被忽视的护城河来源。 如果战略是服务驱动、使命驱动或专家驱动，HRM 通常承重。

### `technology-development` — 技术开发

**Prompt** — 研发、流程技术、产品设计、软件平台。技术在哪里复利？哪里只是入场券？

**Example** — 自有 ML 平台做个性化

**Quality bar** — 研发、流程技术、产品设计、软件平台、自动化和数据基础设施。 问的是：技术在哪里复利，在哪里只是全行业都有的入场券？ 分清**复利型技术**（自有、数据效应、网络效应）和**抹平型技术**（CRM、ERP、云基础设施）。

### `procurement` — 采购

**Prompt** — 投入、服务、容量的采购。买什么、向谁买、如何强化成本或差异化？

**Example** — 认证物料的长期合约

**Quality bar** — 投入物、服务和容量的采购——包括原料、零部件供应商、外包劳动力、云容量和权利 / 许可。 问的是：买什么、向谁买？ 采购战略如何强化成本或差异化？ 采购决策成为 BMC 的关键伙伴和成本结构。

### `inbound-logistics` — 进货物流

**Prompt** — 原料接收与仓储。上游如何稳定、低成本地进入？

**Example** — 零部件准时交付

**Quality bar** — 原料、零部件、容量、数据等投入物的接收、仓储、排程与搬运。 问的是：上游物料和信息能否稳定、按时、以有竞争力的成本进入公司？ 如果进货是成本优势区，把供应商条款和容量合约写到 `business-model-canvas` 的关键伙伴。

### `operations` — 生产运营

**Prompt** — 把投入转为产出。成本与品质差从哪一环产生？

**Example** — 高混合柔性制造

**Quality bar** — 把投入物转化成产出：生产、装配、测试、包装；对服务业而言是交付过程本身。 问的是：成本或品质差到底在哪一环造成？ 运营是大部分成本竞争发生的地方，也是许多差异化战略（品质、定制、速度）的承载点。

### `outbound-logistics` — 出货物流

**Prompt** — 成品仓储与配送。如何以正确时间和成本送达客户？

**Example** — 跨库分拨网络

**Quality bar** — 成品的仓储、履约和向客户的配送。 问的是：公司如何按正确的时间、地点和成本把产品送到客户手中，同时正确地把售后信息流回上游？ 电商 / 实体商品业务里，出货往往是成本熔炉。

### `marketing-sales` — 营销与销售

**Prompt** — 需求生成、品牌、渠道、定价、销售。如何把需求转成营收？

**Example** — 品牌驱动的 D2C，付费与自然组合

**Quality bar** — 需求生成、品牌建设、渠道、定价和销售队伍。 问的是：公司如何触达客户，并以符合单位经济学的成本把需求转成营收？ 每渠道的 CAC 和每细分的 LTV 进入 BMC 的收入来源 + 成本结构。 品牌和渠道承诺进入关键资源 / 渠道。

### `service` — 售后服务

**Prompt** — 安装、维修、支持与照护。如何强化关系并把信息反馈上游？

**Example** — 免费到店维修作为品牌承诺

**Quality bar** — 安装、维修、支持、培训和售后照护。 问的是：服务是在强化客户关系并向上游反馈，还是只是吸收成本？ 如果服务是差异化来源（定义品牌的维修、白手套入门），它必须出现在价值主张里。

### `margin` — 利润

**Prompt** — 客户支付价值与全链路总成本之差。来自成本纪律、差异化溢价、还是兼有？

**Example** — 服务 + 品牌支撑的高端定价

**Quality bar** — 客户支付价值与全链路总成本之差。 问的是：利润到底来自哪——成本纪律、差异化溢价，还是兼有？ 利润是整条链的合成。 如果利润收窄，回头找：是某个成本优势活动在侵蚀？ 是某个差异化活动被复制？ 还是某条联动在漏？

## Colour legend

- `0` — **成本优势活动**: 通过规模、自动化或流程设计相对竞争对手建立可守护成本差的活动。
- `1` — **差异化优势活动**: 通过品质、设计或体验创造客户愿意付溢价的活动。
- `2` — **联动 / 接缝**: 两个活动之间互相加强或互相打架的接缝（如采购品质 → 生产稳定性）。
- `3` — **漏点 / 弱项**: 成本上涨、差异化失效，或能力低于战略意图所要求底线的活动。
- `4` — **能力投入**: 为强化此活动优势需要追加投入的能力或资产。

## Pairs with

这张画布常和这些一起用,做完后引导用户接下去做:

- `business-model-canvas` — 每个价值链活动都对应 BMC 的某个关键活动、关键资源或关键伙伴。不要让 Value Chain 停在图上——把每个活动映射到 BMC 的某个块。
- `design-criteria-canvas` — 把活动级规则（毛利 ≥30%、服务响应 ≤4h、供应商缺陷率 ≤0.1%）编码为下一轮商业模式迭代必须遵守的设计准则。
- `platform-ecosystem-map` — 对于平台 / 多边业务，实体的进货与出货物流几乎消失，技术开发与容量采购成为主导。配合生态地图一起看。
