# 战略读法：Porter 五力模型

瑞士私人银行之所以标记 **Porter 五力模型**，是因为解绑不只是内部运营选择，而是对行业压力的回应：专业基础设施提供商、产品精品店、客户预期变化和监管透明化，都削弱了旧的一体化模式。

行业内竞争来自大量银行都承诺类似的隐私和财富管理能力。供应商力量体现在产品制造商、技术平台和合规基础设施。买方力量随着高净值客户能比较全球管理人、要求独立建议而增强。替代品威胁来自家族办公室、低成本财富平台和专业精品机构。新进入者对完整私人银行来说门槛较高，但对只攻击单一活动的专业玩家来说门槛较低。

Maerki Baumann 的回应是专注客户关系业务，并把其他逻辑外包。Pictet 的回应是保持整合，但投入管理内部权衡。五力模型解释了为什么两种位置都可能自洽：它们是对同一组结构性压力的不同回答。

不要把这个框架简化成“竞争很激烈”。关键洞察是压力进入 BMC 的位置：关键业务、重要合作、成本结构、客户关系和价值主张，都会随着行业解绑而改变。
