# 战略读法：三层增长、BCG 与 McKinsey 7S

P&G Connect + Develop 不只是开放式创新故事。它也展示了成熟企业如何管理未来选项、在组合中分配注意力，并围绕外部创新重新对齐组织。

用**三层增长**看，既有消费品核心是 H1。外部创意、授权和合作技术，在证明客户、技术和商业证据之前都是 H3 选项。当合作方案能整合进品牌平台，并通过 P&G 渠道规模化时，才成为 H2 增长引擎。关键不是把开放式创新当口号，而是让外部选项迁移为内部增长平台。

用 **BCG 增长份额**看，问题是哪些成熟品牌可以为外部搜索能力提供资金，哪些新概念值得规模化。现金牛不应该资助每个想法，而应资助少数能借助 P&G 分销和品牌能力成为明星业务的问题项目。

用 **McKinsey 7S**看，挑战在组织层。Strategy 从“全部内部发明”转向“连接并开发”；Structure 建立外部接口；Systems 筛选、签约并放大合作想法；Shared Values 承认外部发明的合法性；Skills、Style 和 Staff 要奖励协作，而不是只奖励内部拥有。

只有当层级管理、组合投入和组织对齐互相强化时，开放式创新才真正成为战略能力。
