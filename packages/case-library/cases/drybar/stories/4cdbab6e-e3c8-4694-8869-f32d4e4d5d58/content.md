# Strategy lens: Bain Elements of Value

Drybar is not only a Blue Ocean case. The **Bain Elements of Value** lens explains why the concept could command attention without becoming a full-service salon. The value proposition narrows the job to one repeatable occasion: look polished quickly before work, events, travel, or social plans.

Functional value comes from simplification, convenience, reduced effort, and predictable quality. Emotional value comes from confidence, self-expression, and an affordable-luxury ritual. Drybar wins because it over-delivers on a few elements that matter for a narrow occasion and removes the salon services that create complexity.

Read the BMC as the operating system behind these value elements: menu simplicity, standardized styling process, retail-ready spaces, and appointment flow all protect the promise. A VPC version should map pains around inconsistent salons, time waste, and decision fatigue to speed, confidence, and repeatability.

Elements of Value does not prove defensibility. Competitors can copy blowout bars, so the case still needs Blue Ocean and BMC analysis to explain process discipline, brand, location strategy, and repeat operations.
