# 战略读法：Porter 五力模型

AliExpress 是 **Porter 五力模型**案例，因为跨境平台战略受到多方压力塑造，而不只是和其他电商网站竞争。

行业内竞争来自全球平台、本地强者和垂直品类平台对商家、流量和信任的争夺。买方力量很强，因为消费者可以即时比价且切换成本低。供应商力量体现在商家、物流、支付轨道和流量渠道。替代品包括本地零售、品牌直营网店、社交电商和超级 App 电商。完整平台规模的新进入者不容易出现，但跨境细分玩家可以攻击高利润品类。

这个案例说明，平台优势是一套系统：商家密度、价格发现、物流、买家保护、支付和纠纷处理共同降低五力压力。任何一层变弱，买方力量和替代品压力都会迅速上升。
