# Strategy lens: Bain Elements of Value

NovoPen is a strong **Bain Elements of Value** example because the product improves more than functional injection performance. For people managing diabetes, the job includes accuracy, discretion, confidence, routine adherence, and reduced fear of mistakes.

Functional value includes quality, reduces risk, saves time, and simplifies. Emotional value includes reduces anxiety and increases confidence. In some contexts the value can approach life-changing because better medication routines affect daily freedom and perceived control.

A VPC reading should avoid treating the patient as a generic “user.” Separate jobs, pains, and gains for patients, caregivers, and clinicians. The value map should specify how device design, dose memory, portability, and reliability relieve concrete pains rather than merely saying “better device.”

Elements of Value does not replace evidence. Medical-device value must be tested through safety, clinical workflow, adherence, reimbursement, and trust.
