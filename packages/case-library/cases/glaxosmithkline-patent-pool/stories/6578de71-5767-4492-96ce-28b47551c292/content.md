# 葛兰素史克专利池:战略捕获不是许可费,这才是「由内而外」的关键

2009 年 2 月,Andrew Witty 上任 GSK 的 CEO 才半年,就在哈佛医学院做了一场演讲,宣布 GSK 将把与被忽视热带病(NTD)有关的专利汇入池中,向外部研究者开放。整个药企行业当时把这事要么当成异端、要么当成公关动作。《商业模式新生代》*(BMG 2010)* 在 pp. 112-113 中把它选为开放式商业模式中「由内而外」的范式案例。15 年之后回看,BMG 的框架是对的。

## 为什么把 NTD 专利汇入池是理性而不是慈善

大型药企的研发会产出许多对被忽视疾病有潜在活性的化合物 —— 疟疾、利什曼病、血吸虫病、恰加斯病、昏睡病 —— 这些疾病全球大约影响 10 亿人,但其中几乎没有人在经合组织市场。化合物会被筛选,有时候到了早期数据阶段,然后就**被放弃**了。不是因为它们无效,而是因为**没有盈利市场**:GSK 的定价基础设施假设患者能负担每剂 100 美元以上;NTD 影响的人口承担不了。

这些化合物留在 IP 组合里,产出为零。一个专利在 20 年生命周期里申请加维护成本约 5 万美元 —— 所以闲置 IP 不是免费,而是小而持续的现金流出。*(Chesbrough 2006)* 把这种情形精确命名为「由内而外」机会:公司永远不会去商业化的资产,持续累积持有成本,而外部如果能拿到这些 IP,可能确实会从中提取出真实价值。

## 两张 BMC,因为「跨边界的方向」就是模式本身

::canvas[business-model-canvas]{variant="gsk-traditional-pharma"}

传统大型药企 BMC 读起来像每一家重磅药公司。内部科学家发现化合物;临床试验做出疗效证据;监管机构批准;销售代表把它卖出去。「客户细分」和「收入来源」中的粉色便签标出了模型**缺失**的地方:贫困国家患者没有被服务;闲置 IP 的授权收入是零 —— IP 是用来防御的,不是出租。「核心资源」里那张 —— **(对 NTD 有效的闲置化合物 —— 经济上是死重)** —— 命名了即将开始流动的资产。

::canvas[business-model-canvas]{variant="gsk-patent-pool"}

专利池 BMC 在左半边几乎完全没动(核心资源、关键业务、成本结构)—— 由内而外不是改变你怎么做研发,而是改变你**怎么处理那些没赚钱的研发产出**。变化集中在「收入来源」、「客户细分」,尤其是「重要合作」上。

「重要合作」中那些蓝色「跨企业依赖」便签就是新结构:WIPO(联盟运营方)、辉瑞 / 诺华 / Re:Search 联盟中的其他药企、学术界 NTD 研究者、像 DNDi(被忽视疾病药物倡议)和 MMV(疟疾药物风险投资)这样的 NGO。这些在封闭式 BMC 上一个都不存在。**池只有外部被许可方出现时才管用** —— 这正是「跨企业依赖」要捕获的东西。

## 战略捕获到底是什么

*(BMG 2010)* 给出了一个尖锐的判断:许可费收入是**显性**的回收,但不是**重要**的那一个。战略捕获是三件事:

- **在新兴市场积累的声誉。** GSK 的重磅药业务依赖于在它今天还没怎么卖、但 10-20 年后想进入的国家拿到药品审批(印度、巴西、印度尼西亚)。专利池在这些国家的监管层和舆论层积累的声誉与善意,封闭模型做不到。
- **后续研发线索。** 外部研究者使用池中 IP 时,有时会发现 GSK 自己的筛选漏掉的活性。已经有若干池内被许可方发表的研究让 GSK 重新打开过内部研发项目。
- **「自我蚕食」恐惧并未实际发生。** 这是最重要、也最安静的发现。池中的化合物是真正闲置的;开放它们并没有给 GSK 的商业业务培养出竞争对手。注意「关键业务」中那张粉色便签 —— **自我蚕食恐惧管理(内部沟通)** —— 这个恐惧是内部最主要的反对意见,事后证明对**真正**闲置的资产来说,它被严重高估了。

## 哪里会出错(以及 GSK 学到的)

*(Chesbrough 2006)* 警告:由内而外会在三个地方崩掉,而 GSK 在每一个上都做了应对:

- **「自我蚕食」恐惧(对闲置 IP 来说被高估)。** 内部声音说把 IP 授权出去会培养未来的竞争对手。对**真正**闲置的资产 —— 定义为 5 年以上没有内部商业化计划、也没有路线图 —— 这种恐惧从结构上就被高估了。这里需要的纪律是诚实地区分:什么是真闲置,什么只是名义上在战略路线图但永远没动过。GSK 的 800 件专利组成的池,就是这种诚实判断的结果。
- **首批授权定价过低。** 没有授权基准时,首批交易倾向于一次性付费,没有版税也没有股权。「收入来源」中那张粉色便签 —— **(首批授权的定价偏低风险)** —— 是真实存在的。GSK 后来的池交易(2013 年后)加入了版税触发条件,而最早 2010-2012 年的交易没有。
- **战略 / 财务捕获的衡量错配。** 如果只用许可费收入度量「由内而外」项目,这个项目看起来很小(GSK 大概不到每年 5,000 万美元)。这是错的度量方式。正确的度量是声誉 + 准入 + 内部研发线索 + 闲置 IP 持有成本下降。Andrew Witty 在他整个任期向 GSK 董事会为这个项目辩护时,理由始终是战略性的,从来不是财务性的。

## 为什么这个案例与宝洁、InnoCentive 并排放在案例库里

GSK 专利池是开放式中**由内而外**的范式。宝洁 Connect & Develop 是**由外而内**的范式。InnoCentive 是把由外而内**作为服务**提供给众多 Seeker 的连接器平台。三个一起读,开放式的两个子类型结构就清楚了,并且会迫使我们对每个新的「开放」商业点子问一个正确的问题:**价值跨越企业边界时朝哪个方向流动?**

- *外部创意 / IP / 人才流**进**来,在内部使用* → 由外而内(宝洁、GE 绿色创想、乐高 Ideas)。
- *内部闲置 IP 流**出**去,在外部使用* → 由内而外(GSK、IBM 专利授权、特斯拉开放专利)。
- *两个方向都有显著体量* → 双向并行;绝大多数成熟的开放式创新公司(GSK、宝洁、IBM)实际上**两个方向都在跑**。
