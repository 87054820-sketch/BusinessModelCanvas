# 战略读法：Bain 价值元素

CitizenM 之所以标记 **Bain 价值元素**，不是因为它只是“更便宜的酒店房间”。它把功能性简化与现代旅行者的情感和身份价值结合起来。

功能价值包括便利、节省时间、减少精力和稳定质量：自助入住、紧凑房间、可靠设计和核心地段。情感价值包括审美设计、身份符号，以及属于移动化、城市化旅行者群体的感觉。它删除很多客人并不重视的传统服务，再加倍投入目标任务真正需要的价值元素。

VPC 读法要区分两类客人：想要干净、有设计感、高效率落脚点的旅行者，和想要全服务款待体验的旅行者。CitizenM 的 fit 来自明确选择前者，并拒绝后者。

价值元素解释客户吸引力；BMC 和蓝海战略画布解释标准化、空间效率和服务取舍如何让承诺在经济上成立。
