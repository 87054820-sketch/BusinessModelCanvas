# Strategy lens: Bain Elements of Value

Nespresso is tagged with **Bain Elements of Value** because the capsule system sells more than coffee extraction. It layers functional reliability with emotional premium cues and identity value.

Functional elements include quality, saves time, reduces effort, and simplifies: the machine and capsule interface make home espresso repeatable. Emotional elements include design/aesthetics, badge value, and reward: boutiques, membership, packaging, and brand rituals make the recurring capsule purchase feel premium rather than purely utilitarian.

This explains why the same bait-and-hook structure worked only after repositioning. The machine-capsule lock-in created the economic mechanism, but the value elements made customers accept premium pricing and repeat purchase.

The framework has limits. Elements of Value explains willingness to pay; the Free / bait-and-hook pattern and BMC explain the cross-subsidy, patent interface, channel control, and margin capture.
