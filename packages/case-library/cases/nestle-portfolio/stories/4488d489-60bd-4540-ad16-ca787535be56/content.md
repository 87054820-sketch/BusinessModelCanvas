# 战略读法：BCG 增长份额矩阵

## 为什么 BCG 适合这个案例

雀巢比单一产品创业公司更适合做 BCG 示例，因为分析对象是品类、品牌和平台组成的组合。问题不是“雀巢是否创新”，而是成熟现金流业务能否为被选择的增长平台提供资金，同时避免永远保护每一个传统品牌。

::canvas[bcg-growth-share-matrix]{canvasId="19b5241a-d1e0-4cf0-8aaa-de3b5a2b113d"}

现金牛是成熟的全球品牌和重复购买品类，它们为组合提供资金。明星业务需要同时具备品类增长和相对优势。问题业务包括健康科学、植物基等方向：市场有吸引力，但胜出权还没有完全证明。瘦狗是低增长、低适配的业务单元，其资本、管理注意力或品牌位置可能更适合重新配置。

BCG 矩阵应该导向动作，而不是停留在标签。对少数有机会成为明星的问题业务追加投入；收割现金牛，但不要饿死让它们保持韧性的能力；当低增长叠加弱战略适配时，剥离或重新定位。

食品和营养组合不能只用市场增长率和相对份额判断。毛利、监管、渠道权力、品牌信任、供应链韧性和可持续压力，都可能推翻简单象限判断。
