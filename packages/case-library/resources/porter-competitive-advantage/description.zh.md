# 《竞争优势：创造和维持超级绩效》

Porter 1985 年的续作直接建立在《竞争战略》（1980）的结构性基础之上。核心转向：从产业结构 → 企业层面的活动分析。全书核心主张：竞争优势来自企业执行的*一个个离散活动*——而不是把企业当作黑箱，也不是抽象地谈「位置」。

## 价值链（第2章）

价值链把每家企业分解为战略上相关的活动：

**主要活动：**
1. **内部物流** — 接收、存储和分发投入品
2. **运营** — 将投入转化为最终产品
3. **外部物流** — 收集、存储和将产品分发给买方
4. **营销和销售** — 促使和促成购买
5. **服务** — 维持或提升售后产品价值

**支持活动：**
1. **企业基础设施** — 一般管理、规划、财务、法务、质量
2. **人力资源管理** — 招聘、雇佣、培训、薪酬
3. **技术开发** — R&D、流程设计、产品设计
4. **采购** — 购买企业价值链中使用的投入

**利润** — 总价值与完成所有活动的总成本之间的差额。

价值链揭示了*关联*——一项活动的成本或绩效如何影响另一项活动。孤立地优化每项活动往往破坏利润；真正的杠杆在于管理这些关联。PinGarden 的 `porters-value-chain` 画布正是用来可视化地绘制这些关联。

## 成本优势（第3章）

成本优势不在于整个企业——而在于每项离散活动的成本*驱动因素*：
- **规模经济** — 把固定成本分摊到更高产量上
- **学习和外溢** — 累积产量降低单位成本
- **产能利用率** — 接近满产能运转
- **关联** — 价值链内部以及和供应商/渠道的联动
- **相互关系** — 跨业务单元共享活动
- **纵向整合** — 自制还是外购决策
- **时机** — 先发优势或劣势
- **地点** — 要素成本、物流、政府、基础设施
- **制度因素** — 监管、工会、关税

企业通过配置价值链，使得完成所有活动的总成本低于对手来实现成本领先。这*不等同于*「做最便宜的」——这是关于活动层面的经济学。

## 差异化（第4章）

差异化来自企业*如何*执行其价值链活动，而非来自「营销」或「品牌」。每一项活动都是独特性的潜在来源。Porter 区分了：

- **使用标准** — 什么使产品对买方确实更有价值（质量、特性、交付时间、服务）
- **信号标准** — 买方用来*推测*价值的线索（声誉、广告、包装、外观、价格本身）

差异化战略必须缩小信号标准与实际使用标准之间的差距——否则就会变成空心化。

## 技术与竞争优势（第5章）

技术通过影响活动层面的*成本驱动因素*和*独特性驱动因素*而对竞争优势产生作用。Porter 引入了技术价值链的概念：每一项活动都使用某种技术，而产业价值链中技术变革的*模式*决定了技术是否会改变竞争优势。

## 竞争对手选择（第6章）

并非所有竞争对手都是坏的。「好对手」可以：
- 吸收需求波动
- 提升领先者感知的差异化
- 服务领先者不想要的、不具吸引力的细分市场
- 提供成本伞
- 改善与劳动力和政府的谈判地位
- 激励企业内部投入

战略家的任务不是消灭所有对手，而是*选择*哪些可以共存、哪些应该打击。

## 替代分析（第8章）

替代是指某个产品或服务在执行特定买方功能时取代另一个。Porter 提供了迄今最严谨的替代威胁分析框架——在用五力的「替代品威胁」草率概括前，请读完第8章。
