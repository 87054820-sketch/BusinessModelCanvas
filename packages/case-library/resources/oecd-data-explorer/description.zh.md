# 阅读地图 —— OECD 数据浏览器

OECD 数据浏览器适合在战略分析中查找跨经济体可比指标：产业结构、生产率、创新、贸易、创业、就业和政策背景等。

## 它能支持哪些问题

- 哪些国家或产业显示更强的生产率或创新信号？
- 某个市场假设是否与 OECD 可比指标一致？
- 哪些产业或政策趋势应压力测试 BMC？
- 哪些数据可以支持创新指标或情景信号？

## 如何搭配画布

- `business-model-environment`：把产业或政策指标转成外部力量便签。
- `scenario-matrix`：把趋势变化和国家间差异作为情景输入。
- `evidence-scorecard`：把具体 OECD 指标作为支持证据，而不是最终证明。

## 局限性

Data Explorer 是 JavaScript 应用，使用时要仔细核对指标元数据。不同国家、主题和年份的覆盖并不一致。OECD 数据适合做可比指标，但通常不足以直接证明客户层级或公司层级判断。

## 证据纪律

用 OECD 数据约束假设。不要只挑一个指标就下结论，必须核对定义、单位、日期和可比性。
