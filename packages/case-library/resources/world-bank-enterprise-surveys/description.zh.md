# 阅读地图 —— 世界银行企业调查

世界银行企业调查提供企业层级的经营环境证据。它特别适合在 BMC 或五力分析中补充关于融资、基础设施、监管、腐败、竞争、非正规经营、劳动力和企业绩效的证据。

## 它能支持哪些问题

- 这个经济体中的企业最常报告哪些经营约束？
- 融资、基础设施或监管是否会影响成本结构或关键活动？
- 哪些行业力量来自真实运营摩擦，而不只是观点？
- 哪些假设应该转成市场进入的设计准则？

## 如何搭配画布

- `business-model-environment`：把约束转成行业、市场或宏观力量便签。
- `porters-five-forces`：用企业反馈理解供应商力量、买方摩擦、竞争和进入壁垒。
- `business-model-canvas`：把约束连接到关键资源、关键活动、伙伴、渠道或成本。

## 局限性

调查覆盖会随国家、年份、行业、企业规模以及正式/非正式状态变化。企业层级调查比泛泛观点更强，但仍受调查设计和回答情境影响。

## 证据纪律

用调查识别约束和待验证假设。不要把平均值直接当作某个具体业务能否运营的证明。
