# 阅读地图 —— 世界银行数据目录

世界银行数据目录是 PinGarden 中宏观和发展数据的基础入口。当战略判断依赖国家背景、发展指标、公共财政、能源、基础设施或长期环境信号时，优先使用它。

## 它能支持哪些问题

- 目标市场在宏观意义上是否增长？
- 哪些国家或区域指标可以支持环境扫描？
- 哪些外部信号应进入情景矩阵？
- 哪些发展或基础设施约束会影响 BMC？

## 如何搭配画布

- `business-model-environment`：把指标转成趋势、市场、行业或宏观力量便签。
- `scenario-matrix`：用指标变化作为早期信号或不确定性输入。
- `business-model-canvas`：把数据附着到客户、渠道、伙伴、成本和收入假设上。

## 局限性

世界银行指标通常是国家层级并存在时间滞后。它们不一定能描述细分人群、企业行为或快速变化的技术采用。必须记录指标名称、地区、年份，以及该数据是直接指标还是代理指标。

## 证据纪律

宏观指标只能支持假设，不能证明战略一定有效。需要搭配客户、竞争和商业模式证据一起判断。
