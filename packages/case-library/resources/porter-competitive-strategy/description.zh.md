# 《竞争战略：分析产业和竞争对手的技法》

Michael Porter 的第一本书（1980）奠定了现代竞争战略学。在此之前，战略大致等于统兵之道、个案直觉和经验曲线直觉；此后，战略成为一门用可重复框架做结构化分析的学科。

全书论点分为三大部分：

### 第一部分：通用分析技法

**产业结构分析（第1章）**——著名的五力模型：
1. **新进入者的威胁** — 进入壁垒（规模经济、产品差异化、资本需求、转换成本、分销渠道获取、政府政策）
2. **供应商的议价能力** — 供应商集中度、转换成本、前向一体化威胁
3. **买方的议价能力** — 买方集中度、产品标准化程度、后向一体化威胁
4. **替代品或服务的威胁** — 相对性价比、转换成本、买方替代倾向
5. **现有企业之间的竞争** — 产业增长、固定成本、产品差异化、退出壁垒

五力共同决定一个产业的利润潜力。这是每一次商业模式分析都必须立足的结构性基础。

**三种通用战略（第2章）**——三种内部一致的获得高于平均绩效的方式：
- **成本领先** — 广阔市场里的最低成本生产者
- **差异化** — 广阔市场里提供独特价值的产品/服务
- **聚焦** — 服务狭窄细分市场，可以是成本聚焦或差异化聚焦

核心警告："夹在中间"——试图同时追求多于一种通用战略而不做出必要的组织取舍，几乎必然导致低盈利。

**竞争对手分析框架（第3章）**——四个诊断要素：
- **未来目标** — 什么驱动竞争对手
- **当前战略** — 竞争对手正在做什么、能做什么
- **假设** — 竞争对手对自己和产业相信什么
- **能力** — 竞争对手的强项与弱点

### 第二部分：通用产业环境

第9–12章把分析框架应用于五种原型产业结构：
- **碎片化产业** — 没有主导企业（战略：整合）
- **新兴产业** — 新形成或重组（战略：塑造结构）
- **成熟产业** — 增长放缓、经验丰富的买方（战略：成本或差异化）
- **衰退产业** — 需求萎缩（战略：领先、利基、收割、撤资）
- **全球产业** — 跨境竞争（战略：全球 vs. 多国内）

### 第三部分：战略决策

末尾几章处理纵向整合、产能扩张和进入新业务——这些都被框定为改变产业五力的结构性决策，而不只是改变企业自身在其中的位置。

### 本书之于 PinGarden

五力分析是"画 BMC 之前"的步骤——在你动手画商业模式之前，要先理解产业的结构性经济学。用 `business-model-environment` 画布把五力画出来，然后再进入 BMC，看商业模式如何应对这些力量。当五力不利时，没有任何商业模式能持续取得高于平均的回报。
