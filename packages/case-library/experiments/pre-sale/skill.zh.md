# 预售 — AI Agent 速查

**什么时候推荐。** B2C / prosumer,有单一挂牌价($20-$500)。Smoke Test 通过、下一个 gate 是 "注册的人会真付钱吗?"。用户能在承诺时限内交付。

**证据强度。** 强。真金白银能筛掉客气意向 —— 相比口头声明,行为变化最大。

**样本。** 1000-5000 目标访问,转化阈值 + 收入阈值,跑前都定好。

**准备 / 执行。** 周(页面 + 支付 + 广告创意 + 交付计划)/ 周。

**能力。** `marketing-launch`、`payment-processing`、`customer-communication`。

**决策树。**

- 时限内交付不了? → 不推荐。名誉燃烧弹。
- B2B 走采购? → 不推荐。Letter of Intent。
- 企业销售一对一定价(无挂牌)? → 不推荐。Letter of Intent。
- 监管类目(医疗/金融)? → 顶回去;先过法务。

**反模式。**

- 承诺超过能发货的。
- 无退款政策。
- 无目标流量 = 好奇退款。
- 把收入当利润(是带交付承诺的运营资本)。

**跨画布。** 验证 `business-model-canvas`(在那个价位的收入流 + 客户细分)、`value-proposition-canvas`(哪种价值结晶成了真付费)。
