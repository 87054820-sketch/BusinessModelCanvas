# Bain Elements of Value

## Select this framework when

- The value proposition is vague or overuses words like convenient, premium, simple, or delightful.
- The user needs to deepen VPC, JTBD, empathy, or customer journey work.

## Use these canvases

1. `value-proposition-canvas` — map selected value elements into gain creators and pain relievers.
2. `jobs-to-be-done` — explain the situation and job behind the value.
3. `empathy-map` — test emotional and social value from the customer view.
4. `customer-journey` — locate where value appears or breaks.

## Questions to ask

- Which segment and situation are we analyzing?
- Which functional values are proven?
- Which emotional or higher-level values are evidenced?
- Which few elements should define the proposition?

## Red flags

Do not claim all 30 elements. Do not treat elements as marketing copy. Do not add higher-level value before functional value works.
