# 贝恩价值元素

贝恩价值元素把客户价值拆成 30 个元素，分布在功能、情感、改变人生和社会影响四个层级。它帮助团队从泛泛的痛点和收益，深入到更具体的价值主张。

## 何时使用

当团队已经有客户细分，但价值主张仍然很空泛，例如“方便”“高端”“简单”“令人愉悦”时使用。这个框架可以帮助你明确客户到底重视哪些价值元素，以及哪些元素应该被提升、组合或验证。

不要用它声称自己提供所有价值元素。强价值主张通常是围绕某个细分人群，选择少数真正重要的元素打透。

## 如何映射到 PinGarden 画布

- `value-proposition-canvas` 把价值元素映射为痛点缓解、收益创造和产品/服务。
- `jobs-to-be-done` 解释某个元素为什么在特定情境中重要。
- `empathy-map` 从客户视角检查情感和社会价值。
- `customer-journey` 找到价值在体验中的哪个阶段被创造或丢失。

## 使用流程

1. 选择一个客户细分和一个使用情境。
2. 识别当前已经交付的功能价值元素。
3. 只有在有证据时，才加入情感、改变人生或社会影响元素。
4. 将当前方案与替代方案比较。
5. 把选中的元素映射到 VPC 的收益创造和痛点缓解。
6. 用客户旅程定位每个价值元素出现在哪个阶段。
7. 测试客户是否真的识别并愿意为这些元素付费。

## 示例中看什么

- `nespresso` 组合了质量、感官吸引力、减少努力、身份象征和持续获取。
- `drybar` 围绕体验、减少努力和自信感缩窄服务。
- `stitch-fix` 组合了多样性、个性化、减少努力和信心。
- `citizenm-hotels` 聚焦简化旅行、设计和便捷获取。
- `novo-nordisk-novopen` 在医疗情境中体现功能易用和减轻焦虑。

## 常见误用

- 选择过多元素，导致价值主张失焦。
- 把价值元素当成营销说法，却没有客户证据。
- 忽略客户在什么情境下重视该元素。
- 把功能便利误当成情感安心。
- 在基础功能价值尚未成立前，就声称提供高层级价值。
