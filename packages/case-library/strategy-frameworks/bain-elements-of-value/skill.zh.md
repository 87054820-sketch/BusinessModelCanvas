# 贝恩价值元素

## 何时选择

- 价值主张很空泛，反复使用“方便”“高端”“简单”“愉悦”等词。
- 用户需要深化 VPC、JTBD、共情图或客户旅程。

## 使用画布

1. `value-proposition-canvas` —— 把选中的价值元素映射到收益创造和痛点缓解。
2. `jobs-to-be-done` —— 解释价值背后的情境和任务。
3. `empathy-map` —— 从客户视角验证情感和社会价值。
4. `customer-journey` —— 定位价值在哪些阶段出现或断裂。

## 关键问题

- 分析哪个客户细分和情境？
- 哪些功能价值已经被证明？
- 哪些情感或高层级价值有证据？
- 哪几个元素应定义这个价值主张？

## 红旗

不要声称 30 个元素全都有。不要把元素当营销文案。基础功能价值未成立前，不要急着添加高层级价值。
