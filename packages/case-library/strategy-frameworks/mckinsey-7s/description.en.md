# McKinsey 7-S Framework

McKinsey 7-S is an organization-alignment framework. It checks whether **Strategy, Structure, Systems, Shared Values, Style, Staff, and Skills** reinforce one another. The method is useful because strategy rarely fails only because the strategy statement is wrong; it often fails because the organization cannot execute it.

## When to use

Use it when a company has a strategy but the operating model, culture, talent, or capabilities appear misaligned. It is strong for transformation, post-merger integration, innovation governance, and scaling new business models into the core.

Do not use it as a generic culture survey. If the question is customer value, use VPC or Elements of Value. If the question is portfolio timing, use Three Horizons.

## How it maps to PinGarden canvases

- `innovation-culture-map` captures outcomes, behaviors, enablers, and blockers that reflect 7-S misalignment.
- `design-criteria-canvas` turns alignment needs into decision rules for operating model design.
- `business-model-canvas` shows what the organization must support: value proposition, channels, activities, resources, partners, revenue, and cost.

## Workflow

1. Start with the strategy: what choice must the organization execute?
2. Map structure and systems that currently support or block the choice.
3. Map staff and skills gaps that make execution fragile.
4. Identify shared values and leadership style that reinforce or contradict the strategy.
5. Translate gaps into behavior and mechanism notes on `innovation-culture-map`.
6. Write design criteria for what must change.
7. Recheck whether the BMC can realistically be delivered by the current organization.

## What to notice in examples

- `bosch-accelerator` shows the organizational mechanics needed to protect exploration.
- `procter-gamble-cd` shows external innovation only works when systems and capabilities support it.
- `ping-an-group` shows group-level strategy requires structure, skills, and systems across financial and technology units.
- `patagonia` shows values and operating choices reinforcing the strategic position.

## Common misuses

- Listing seven words without diagnosing the interactions between them.
- Treating shared values as slogans instead of observable decisions.
- Ignoring systems, incentives, and governance because culture feels easier to discuss.
- Using 7-S after the strategy is already fixed, instead of using it to test executability.
- Failing to connect organization issues back to a concrete business model.
