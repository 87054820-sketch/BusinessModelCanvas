# Blue Ocean Strategy — AI guide

Use this framework when the user needs to compare a company against an industry playbook, design a differentiated value curve, or ask whether an idea can create uncontested market space.

## Fill order

1. Read the case or user idea and identify the **red-ocean industry norm**.
2. Define 5–10 **factors of competition** customers actually compare.
3. Plot incumbents and the focal company on the same factors using `blue-ocean-strategy-canvas`.
4. Derive ERRC from the curve: **Eliminate / Reduce / Raise / Create**.
5. Translate the new value curve into BMC implications: value proposition, segments, channels, key activities, resources, partners, revenue, and cost.

## Quality bar

- The blue-ocean curve must have a different shape, not just higher scores.
- Every score needs a source or explicit assumption.
- Explain the noncustomers or underserved buyers unlocked by the move.
- Pair every Strategy Canvas case with at least one BMC story.

## Anti-patterns

- Do not classify Blue Ocean Strategy as a business-model pattern.
- Do not tag a case as blue-ocean just because it is innovative.
- Do not write ERRC before plotting the competitive factors.
- Do not copy long book passages; write original summaries with citations.
