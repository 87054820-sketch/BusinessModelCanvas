# 蓝海战略 — AI 使用指南

当用户需要对比某家公司与行业常规做法、设计差异化价值曲线，或判断一个想法是否可能开创无人竞争的市场空间时，使用这个框架。

## 填写顺序

1. 先读案例或用户想法，识别**红海行业常规**。
2. 定义 5–10 个客户真实比较的**竞争因素**。
3. 用 `blue-ocean-strategy-canvas` 把在位者和目标公司画在同一组因素上。
4. 从曲线差异推导 ERRC：**剔除 / 减少 / 提升 / 创造**。
5. 把新价值曲线翻译回 BMC：价值主张、客户细分、渠道、关键活动、关键资源、伙伴、收入和成本。

## 质量标准

- 蓝海曲线必须形状不同，而不是所有分数都更高。
- 每个评分都要有来源或明确假设。
- 说明被打开的非顾客或未充分服务买方是谁。
- 每个战略画布案例至少配一篇 BMC 相关故事。

## 反模式

- 不要把蓝海战略归类为商业模式 pattern。
- 不要因为一个案例“很创新”就打蓝海标签。
- 不要先写 ERRC，再倒推竞争因素。
- 不要大段复制书籍内容；应写原创摘要并保留引用。
