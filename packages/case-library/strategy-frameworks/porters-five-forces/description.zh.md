# 波特五力模型

波特五力模型通过五种竞争压力分析行业结构：现有竞争者之间的竞争、买方议价能力、供应商议价能力、新进入者威胁和替代品威胁。它的重点是在选择商业模式位置之前，先理解利润压力来自哪里。

## 何时使用

当团队需要理解某个行业为什么有吸引力或没有吸引力、利润率为什么受压，或新商业模式必须中和哪种力量时使用。它适合进入市场前、重新定位商业模式时、或压力测试环境假设时使用。

不要把它当成竞争对手列表。竞争对手只是五种力量之一。也不要用它直接设计客户价值；行业压力明确后，再搭配 VPC 或 BMC。

## 如何映射到 PinGarden 画布

- `business-model-environment` 捕捉 BMC 周围的行业力量、市场力量、趋势和宏观力量。
- `business-model-canvas` 展示公司如何通过活动、资源、伙伴、渠道、成本和收入进行回应。
- `design-criteria-canvas` 把行业压力转成更有防御性的商业模式设计准则。

## 使用流程

1. 定义行业边界和分析单元。
2. 识别直接竞争以及竞争基础。
3. 分别分析买方力量和供应商力量。
4. 识别新进入者和进入壁垒。
5. 识别用不同方式完成同一任务的替代品。
6. 把每种力量转成 BMC 压力点。
7. 定义战略回应：避开、重塑、防守、合作、差异化或改变模式。

## 示例中看什么

- `swiss-private-banking` 展示解绑压力如何改变行业结构。
- `mobile-telco-unbundling` 展示基础设施、产品和客户关系的压力。
- `carvana` 展示新模式如何改变二手车零售中的竞争和替代。
- `aliexpress` 展示平台力量、供应商接入、买方选择和跨境替代。
- `patagonia` 展示差异化和价值观如何降低直接价格竞争。

## 常见误用

- 把五力模型当成竞争对手清单。
- 因为替代品不在正式行业内就忽略它。
- 看到市场增长就认为行业有吸引力，却不看权力和竞争。
- 只罗列五种力量，不转成商业模式含义。
- 把行业吸引力和公司能力混为一谈。
