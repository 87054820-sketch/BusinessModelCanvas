# 何时使用价值主张画布

价值主张画布（Value Proposition Canvas, VPC）是商业模式画布的插件工具：它把商业模式中的「客户细分」与「价值主张」放大，拆成 **客户画像** 与 **价值地图** 两侧，帮助你判断：**你提供的东西，是否真的匹配某个具体客户细分最重要的任务、痛点和收益？**

它不是一次性填写的静态模板，而是把价值主张假设写清楚、排优先级、拿去验证和迭代的工具。

## 适合的场景

- 你已经有一个候选客户细分，想判断产品/服务是否真正对得上。
- 你想解释一个功能「为什么值得做」，而不只是「它是什么」。
- 团队正在围绕一个想法发力，但还没说清楚客户真正要完成的任务。
- 你要准备客户访谈、着陆页测试或购买意愿实验，需要一个可验证的假设。
- 你要比较多个价值主张方向，找出最有差异化和商业潜力的一版。

## 不适合的场景

- 你还没有明确的候选客户细分——先用更宽的客户探索工具。
- 你想刻画整个业务系统——用商业模式画布更合适。
- 你只是优化一个已被充分验证的成熟功能——VPC 的探索框架可能偏重。
- 你只想写一句营销口号——先用 VPC 找到 fit，再用 Ad-Lib 价值主张速写收敛表达。
