# 何时使用价值主张画布

当你需要深入理解客户真正想要什么，并设计与之匹配的供给时，请使用价值主张画布。当你分析的对象是单一客户群体及其价值主张时，它是商业模式画布的完美搭档。

## 适用场景

- 你正在设计新产品、服务或功能，需要确保它创造真正的客户价值。
- 你想超越功能和技术，理解特定客户群体的任务、痛点和收益。
- 你的团队在没有共享客户理解的情况下争论想法。
- 你需要测试你的价值主张是否真正契合客户所关心的内容。
- 你想让产品、营销和销售围绕清晰的客户画像对齐。

## 不适用场景

- 你需要绘制整个商业模式，而不仅仅是一个价值主张 —— 请改用**商业模式画布**。
- 你的分析跨越多个需求不同的客户群体 —— VPC 一次只聚焦一个群体。
- 你正在解决没有外部客户影响的内部运营问题。
