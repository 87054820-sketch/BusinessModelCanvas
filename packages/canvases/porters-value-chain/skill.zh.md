# 波特价值链

## 一句话总结

9 类活动把公司拆开。任务是：找出竞争优势在哪一环形成——成本、差异化或两者兼有；并指出**联动**在哪里相互加强、在哪里相互打架。

## 填写顺序

1. **进货 → 服务**（从左到右）先填。把运营骨架画上去，再讨论优势。
2. **支持活动**（企业基础设施 → 采购）。它们看似"无聊"，但通常是最难复制的护城河。
3. 回过头**给每个活动标颜色**：黄=成本优势、蓝=差异化、紫=联动、粉=漏点。
4. **利润**区最后填——汇总公司到底靠什么挣钱。

## 跨块不变量

- 联动是真正的优势所在。生产运营 ↔ 采购 之间的一张紫色联动 sticky，比两个填满了"特性"的格子更值钱。
- 支持活动不是可选项。一家 HRM 薄弱的公司，无论营销 sticky 怎么写都做不出服务导向的战略。
- 同一活动同时有成本 + 差异化优势是可能的（如 Patagonia 的可持续采购既是成本纪律也是品牌差异化），但少见——遇到时用双色标记。

## 反模式

- 通用流程图，没有做优势评估。
- 单独评估每个活动，跳过联动颜色。
- 因为"无聊"跳过支持活动。
- 平台 / 服务业务：假装进货和出货物流还重要——它们已经接近零；技术开发 + 容量采购才是主导。
- 分析停在画布上，没有翻译成 BMC 的 KA / KR / KP。

## 语气

活动级具体性。「生产运营」本身不够——写明是**哪种**生产（高混合柔性制造？7×24 自动化云服务？）。数字和动词胜过形容词。
