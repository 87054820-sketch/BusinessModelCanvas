# Strategy Canvas (Blue Ocean)

> Plot one value curve per company across the 5–15 factors the industry
> competes on. The shape of each curve is the strategy.

The Strategy Canvas (W. Chan Kim & Renée Mauborgne, *Blue Ocean
Strategy*) compresses an industry's competitive logic into one picture:
a horizontal axis of **competitive factors**, a vertical scale of
**offering level** (from "Low" to "High"), and one **value curve** per
company plotted across them.

A "blue ocean" curve **focuses** (it doesn't try to win on every factor),
**diverges** (it doesn't follow the industry's average shape), and reads
like a **compelling tagline** at a glance.

## How to use this canvas

1. **Add factors.** What does this industry compete on today? Price,
   service speed, breadth of selection, in-store experience, brand
   prestige — whatever your customers explicitly evaluate. Edit the
   factor list in the right inspector. Six starters are pre-populated.
2. **Plot the incumbents.** Add 1–3 lines for the dominant players or
   the industry average; rate each factor 0–5.
3. **Plot yourself.** Add a line for your own offering. Compare shapes.
4. **Run the four actions.** For each factor ask:
   - **Eliminate** — which factors should be removed entirely?
   - **Reduce** — which should be pushed below industry standard?
   - **Raise** — which should be raised well above standard?
   - **Create** — which factors should be added that the industry has
     never offered?
5. **Annotate with pins.** Drop a pin near a curve point to write a
   one-line note ("Yellow Tail's edge here"; "ELIMINATE — tannin
   complexity"). Pins can anchor to a specific (line, factor) pair so
   they follow the line as you re-score.
6. **Compare divergence.** A blue ocean curve looks visibly *different*
   from the incumbents'. If your curve hugs theirs, you're competing on
   the same red ocean — go back to step 4.
