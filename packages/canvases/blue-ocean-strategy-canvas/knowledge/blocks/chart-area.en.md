# Chart Area

This module is the Strategy Canvas itself: competitive factors on the horizontal axis, offering level on the vertical axis, and one value curve per competitor or strategic option. The chart is not a feature table. It is a visual argument about how an industry competes today and how a new strategy could change the basis of value.

## Module Purpose

Use the chart to make three things visible: which factors the industry invests in, how similar competitors look to buyers, and where a new curve can focus, diverge, and tell a compelling story. The point is not to score perfectly; the point is to see strategic shape.

## Good Answer vs Weak Answer

A good chart uses customer-facing factors, plots current players before the proposed move, and shows a visibly different curve backed by ERIC logic. A weak chart lists internal capabilities, gives everyone similar scores, or draws a heroic curve without explaining what will be eliminated, reduced, raised, or created.

## Common Pitfalls

- Using factors customers cannot perceive, such as internal process metrics.
- Treating "blue ocean" as simply premium, cheap, or novel.
- Drawing the future curve before understanding incumbents and noncustomers.
- Forgetting to test whether the new curve works in BMC revenue, cost, partners, and resources.

## Sticky Examples

- "Factor: ease of selection. Yellow Tail high; traditional wine low."
- "Eliminate: expert vocabulary and aging rituals."
- "Create: guided setup for first-time business buyers."
- "Noncustomer insight: people avoid the category because choosing feels embarrassing."

## Expected Output

The output should be a readable current-state curve, one proposed to-be curve, and a short set of ERIC notes that explain why the shape changes. If the output does not change BMC choices about customer segment, value proposition, revenue, cost, partners, or resources, the chart is still decorative.

## Related Cases

Yellow Tail shows how a new curve can remove wine-snob factors and create easy selection. Cirque du Soleil shows category reconstruction by eliminating animals and raising artistic experience. Nintendo Wii shows how noncustomers can become central when the value curve shifts away from raw hardware performance.

## Transfer Questions

- Which competitive factor exists mainly because the industry inherited it?
- Which factor could be reduced without hurting the target noncustomer?
- If this curve is true, which BMC block changes first: value proposition, customer segment, channel, revenue, cost, partners, or resources?
- What evidence would convince you that buyers actually value the created factor?
