# How to use

1. Start from external signals found in environment scanning.
2. Choose two independent uncertainties that would most affect strategic choices.
3. Name both end states for each uncertainty and combine them into four scenarios.
4. For each scenario, write market structure, customer behavior, key risks, and BMC pressure points.
5. Summarize robust moves and early warning signals at the bottom.

# Anti-patterns

- Writing optimistic, base, pessimistic, and disaster versions.
- Creating vivid scenarios that do not change strategy.
- Ignoring early signals, so scenario planning cannot be updated over time.
