# 情景 3

情景 3 常常帮助团队发现隐藏假设。不要把它简单写成“坏”。要问：当两条轴这样组合时，会出现哪些新赢家、替代方案、约束或非顾客？

## 好答案 vs 弱答案

好答案会暴露一条不同的价值路径或失败路径。弱答案只写收入下降、成本上升或客户延迟。

## 常见误区

- 用情景来证明团队现有战略是对的。
- 忽略非顾客和相邻市场。
- 情景写完后和决策没有连接。

## 便签示例

- “监管收紧 + 平台碎片化：可信中介变得有价值。”
- “预算增长 + 模型商品化：差异化转向工作流数据和客户成功。”

## 迁移问题

- 在这个未来里，什么变得更有价值，而当前模型投入不足？
- 哪个实验能测试最意外的含义？
