# Scenario 4

Scenario 4 completes the matrix and should force comparison across all futures. The question is not whether this is likely; the question is whether it is plausible enough to expose a blind spot or create a strategic option.

## Good Answer vs Weak Answer

A good answer produces implications that differ from the other three scenarios. A weak answer only changes adjectives while keeping the same market logic.

## Common Pitfalls

- Making the fourth scenario a leftover.
- Ignoring operational constraints such as talent, cash cycle, compliance, or supply.
- Failing to name early signals.

## Sticky Examples

- "Restrictive regulation + platform concentration: access is controlled by licensed giants; niche trust plays survive."
- "Budgets freeze + proprietary models scarce: only urgent regulated use cases can pay."

## Transfer Questions

- What would you regret not preparing if this future arrived?
- Which BMC block needs an option rather than a firm commitment?
