# 生产运营

把投入物转化成产出：生产、装配、测试、包装；对服务业而言是交付过程本身。问的是：成本或品质差到底在哪一环造成？运营是大部分成本竞争发生的地方，也是许多差异化战略（品质、定制、速度）的承载点。

## 例子

- "按周重新平衡排程的高混合柔性制造。"
- "跨 3 地的 7×24 云服务运营，五个 9 的 SLA。"
- "标准化咨询销售剧本，每次客户通话都打质量评分。"

## 接下来

对每个子活动标：规模经济（是 / 否）、能力稀缺性（通用 / 专家 / 自有）、当下能力强度（1-5）。差异化的运营应出现在价值主张里；成本优势的运营应出现在 BMC 的成本结构和关键资源里。
