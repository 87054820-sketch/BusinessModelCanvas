# 人力资源管理

招聘、培训、薪酬、文化、留任和领导力发展。问的是：哪些能力扎根在人身上？人才管线在哪个环节决定战略的成败？

## 例子

- "高级工程师的专科招聘 + 学徒管线；18 个月内部晋升路径。"
- "使命驱动文化 + 公开 ESG 承诺；流失率降到行业中位数一半。"
- "跨 3 个时区的分布式工程师梯队；24 小时交接手册。"

## 接下来

HRM 是最容易被忽视的护城河来源。如果战略是服务驱动、使命驱动或专家驱动，HRM 通常承重。把人才假设映射到 BMC 的关键资源（人的一面），并明确标注：当 Ansoff 或 Three Horizons 动作要求公司还没有的能力时——这是一道明确的红线。
