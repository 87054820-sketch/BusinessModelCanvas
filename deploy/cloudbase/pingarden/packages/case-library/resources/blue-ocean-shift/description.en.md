# Reading map — Blue Ocean Shift

Read this after *Blue Ocean Strategy*. The first book is mainly a strategy toolkit; this one is a transformation manual for moving a real team from “we understand differentiation” to “we can make a market-creating move with confidence.”

## What problem it solves

- A team draws a new strategic profile, but the organization still acts through old metrics and boundaries.
- Blue ocean work becomes a top-down idea rather than a participatory process.
- Noncustomer insight is collected but does not shape decisions.
- New strategy lacks staged validation, so the team either overcommits or keeps debating.

## Contents / reading map

1. **Get started** — choose a scope, assemble the team, and build shared language for market creation.
2. **Understand where you are** — use Strategy Canvas to show why the current model is stuck in red-ocean competition.
3. **Imagine where you could be** — study noncustomers, buyer utility, and barriers to demand.
4. **Find blue ocean options** — create several possible moves instead of betting on one idea.
5. **Make the shift** — use fair process, evidence, and staged commitment to reduce organizational resistance.

## How it maps to PinGarden

- `blue-ocean-strategy-canvas` — current and target value curves.
- `customer-journey` — buyer utility, barriers, and noncustomer drop-off points.
- `design-criteria-canvas` — converts strategic choices into design criteria.
- `experiment-canvas` — validates new demand, touchpoints, and payment logic.
- `business-model-canvas` — turns the blue ocean option into an operating model.

## What to read first

- For **red-to-blue transformation**, read the process chapters and turn each step into a workshop.
- For **noncustomer research**, focus on noncustomer tiers, buyer utility, and adoption barriers.
- For **internal alignment**, read fair process and confidence-building sections.
- For **new venture approval**, connect candidate moves to `experiment-canvas` before scaling investment.

## Common misreadings

- **It is not only a case-book sequel.** Its value is the process and human side of transformation.
- **Finding noncustomers is not enough.** Their barriers must be converted into value, price, cost, and adoption logic.
- **Consensus is not built by presentations alone.** Shared facts, shared choices, and evidence create confidence.
