# Reading map — Blue Ocean Strategy

Read this after the team already has a first BMC or value proposition. The book is not simply about “finding a market without competition”; it provides tools for redrawing competitive factors and creating a new value curve.

## What problem it solves

- Teams compare competitor features while accepting the same industry boundaries.
- Differentiation becomes “more features” or “more premium” without changing cost and buyer value together.
- Strategy discussions lack a visual way to see where the industry is over-investing.
- Demand creation focuses only on existing users and ignores noncustomers.

## Contents / reading map

1. **Value innovation** — reshape buyer value and cost structure together.
2. **Strategy Canvas** — map competitive factors and value curves across players.
3. **Four Actions Framework** — Eliminate, Reduce, Raise, Create.
4. **Six Paths** — reconstruct market boundaries through alternatives, strategic groups, buyer chains, complements, functional/emotional orientation, and time.
5. **Reach beyond existing demand** — analyze noncustomers instead of only segmenting current customers.
6. **Execution risks** — address cognitive, resource, motivational, and political hurdles.

## How it maps to PinGarden

- `blue-ocean-strategy-canvas` — the main visual tool for factors and value curves.
- `business-model-canvas` — translates a new value curve back into resources, activities, cost, and revenue.
- `value-proposition-canvas` — checks whether the new curve maps to real customer jobs, pains, and gains.
- `design-criteria-canvas` — turns Eliminate/Reduce/Raise/Create into design constraints.

## What to read first

- For **competitive analysis**, start with Strategy Canvas and avoid feature-table thinking.
- For **strategic innovation**, read Four Actions and Six Paths to break industry defaults.
- For **new product positioning**, read noncustomers and buyer utility to find unserved demand.
- For **organizational adoption**, read execution sections and focus on how teams accept a new curve.

## Common misreadings

- **Blue ocean does not mean no competition forever.** It means changing the factors that make current competition relevant.
- **New features are not automatically value innovation.** They must create buyer value without simply adding cost and complexity.
- **Strategy Canvas is not only a presentation chart.** It is a decision tool for what to eliminate, reduce, raise, and create.
