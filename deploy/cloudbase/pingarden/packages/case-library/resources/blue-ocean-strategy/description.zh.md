# 《Blue Ocean Strategy / 蓝海战略》阅读地图

这本书适合在团队已经有一版 BMC 或价值主张之后阅读。它不是让你“找一个没有竞争的市场”这么简单，而是用系统工具重画竞争因素，创造新的价值曲线。

## 这本书解决什么问题

- 团队只会横向比较竞品功能，默认产业边界和竞争规则不变。
- 差异化被理解为“多做一点”或“高端一点”，没有同时降低成本和提升买方价值。
- 战略讨论缺少可视化工具，无法看清行业都在投资哪些竞争因素。
- 需求创造只盯现有用户，没有认真分析非顾客。

## 目录/阅读地图

1. **价值创新**：蓝海战略的核心不是技术创新，而是在买方价值和成本结构之间重新组合。
2. **战略布局图 / Strategy Canvas**：画出现有竞争因素和各玩家的价值曲线，找到同质化区域。
3. **四步动作框架**：Eliminate、Reduce、Raise、Create，用来重构价值曲线。
4. **六条路径**：从替代行业、战略群体、购买者链、互补产品、功能/情感导向、时间趋势等角度重建市场边界。
5. **超越现有需求**：分析非顾客，避免只在现有细分市场里做微调。
6. **落地与组织风险**：处理认知、资源、动力和政治阻力，让战略不只停在图上。

## 和 PinGarden 画布怎么对应

- `blue-ocean-strategy-canvas`：画行业竞争因素和价值曲线，是本书最核心的可视化入口。
- `business-model-canvas`：把新的价值曲线转回关键资源、关键活动、成本结构和收入来源。
- `value-proposition-canvas`：检查新价值曲线是否真的对应客户任务、痛点和收益。
- `design-criteria-canvas`：把“消除/减少/提升/创造”转成设计约束。

## 建议重点读哪些章节

- 如果你在做**竞品分析**：先读 Strategy Canvas，避免只列功能表。
- 如果你在做**战略创新**：重点读四步动作框架和六条路径，把行业默认条件逐一拆开。
- 如果你在做**新产品定位**：读非顾客和买方效用相关内容，找出为什么现有人群之外的人没有被服务。
- 如果你在做**组织推动**：读执行部分，关注如何让团队接受新价值曲线。

## 常见误读

- **误读 1：蓝海等于没有竞争。** 更准确地说，是通过改变竞争因素，让原有竞争维度失效。
- **误读 2：只要创造新功能就是蓝海。** 如果新功能同时提高复杂度和成本，却没有显著提升买方价值，就不是价值创新。
- **误读 3：战略布局图只是汇报图。** 它应该用于讨论取舍：哪些因素要砍掉、降低、提高或创造。

## 下一步可以打开

- 想看清行业竞争轮廓：打开 `blue-ocean-strategy-canvas`。
- 想把新曲线落到商业逻辑：打开 `business-model-canvas`。
- 想验证价值是否成立：打开 `value-proposition-canvas` 和 `experiment-canvas`。
- 想参考案例：查看 `cirque-du-soleil`、`yellow-tail`、`qb-house`、`nintendo-wii`。
