# 阅读地图 —— WIPO 全球创新指数

WIPO 全球创新指数基于约 80 项来自国际公共和私营来源的指标，对接近 140 个经济体的创新表现进行排名。它适合用于创新生态背景分析，而不是证明某家公司一定有创新能力。

## 它能支持哪些问题

- 哪些国家或地区有更强的创新投入和创新产出？
- 哪些创新生态信号应影响情景规划？
- 哪些市场可能支持 H2/H3 增长押注？
- 哪些证据可以支持生态层面的创新指标？

## 如何搭配画布

- `business-model-environment`：添加创新基础设施、政策、教育、投资和知识创造信号。
- `scenario-matrix`：把创新投资、技术采用或集群信号作为不确定性或早期信号。
- `portfolio-map` 和 `three-horizons-map`：支持未来押注是否有生态顺风的假设。
- `evidence-scorecard`：把生态证据和客户/商业模式证据分开记录。

## 局限性

GII 是经济体层级指数。排名可能掩盖国内差异、行业差异和企业执行差距。使用前应核对指标定义和年份覆盖。

## 证据纪律

用 GII 描述创新生态和比较市场。不要仅凭国家排名推断公司级创新能力。
