# Reading map — WIPO Global Innovation Index

WIPO Global Innovation Index ranks innovation performance across close to 140 economies using around 80 indicators from public and private international sources. It is useful for innovation ecosystem context, not for proving that a specific company is innovative.

## What it helps answer

- Which countries or regions have stronger innovation inputs and outputs?
- Which innovation ecosystem signals should influence scenario planning?
- Which markets may support H2/H3 growth bets?
- Which evidence can support innovation metrics at ecosystem level?

## How to use with canvases

- `business-model-environment`: add innovation infrastructure, policy, education, investment, and knowledge-creation signals.
- `scenario-matrix`: use innovation investment, technology adoption, or cluster signals as uncertainties or early signals.
- `portfolio-map` and `three-horizons-map`: support assumptions about whether future bets have ecosystem tailwinds.
- `evidence-scorecard`: record ecosystem evidence separately from customer or business-model evidence.

## Limitations

GII is an economy-level index. Rankings can hide within-country variation, industry differences, and firm-level execution gaps. Indicator definitions and year coverage should be checked before using the data.

## Evidence discipline

Use GII to describe ecosystem context and compare markets. Do not infer company-level innovation capability from country rank alone.
