# Reading map — OECD Data Explorer

OECD Data Explorer is useful when a strategy analysis needs comparable indicators across advanced and partner economies: industry structure, productivity, innovation, trade, entrepreneurship, employment, and policy context.

## What it helps answer

- Which countries or sectors show stronger productivity or innovation signals?
- Is a market assumption consistent with comparable OECD indicators?
- Which industry or policy trends should pressure-test the BMC?
- What evidence supports innovation metrics or scenario signals?

## How to use with canvases

- `business-model-environment`: turn sector or policy indicators into external-force stickies.
- `scenario-matrix`: use trend changes and cross-country divergence as scenario inputs.
- `evidence-scorecard`: attach a named OECD indicator as supporting evidence, not final proof.

## Limitations

The Data Explorer is a JavaScript application and indicator metadata should be checked carefully. Coverage varies by country, topic, and year. OECD data is strong for comparable indicators, but not always enough for customer-level or company-level claims.

## Evidence discipline

Use OECD data to discipline assumptions. Do not cherry-pick one indicator without checking definition, unit, date, and comparability.
