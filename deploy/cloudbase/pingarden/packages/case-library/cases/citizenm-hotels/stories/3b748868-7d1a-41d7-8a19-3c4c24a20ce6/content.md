# Strategy lens: Bain Elements of Value

CitizenM is tagged with **Bain Elements of Value** because its value proposition is not simply “cheaper hotel rooms.” It combines functional simplicity with emotional and identity value for modern travelers.

Functional value includes convenience, saves time, reduces effort, and predictable quality: self check-in, compact rooms, reliable design, and central locations. Emotional value includes design/aesthetics, badge value, and a sense of belonging to a mobile, urban traveler identity. The hotel removes traditional services that many guests do not value, then over-invests in the elements that define the target job.

A VPC reading should separate the traveler who wants a clean, stylish, efficient base from the traveler who wants full-service hospitality. CitizenM's fit depends on choosing the first job clearly and rejecting the second.

Elements of Value explains customer pull; the BMC and Blue Ocean canvas explain how standardization, space productivity, and service trade-offs make the promise economically viable.
