# 雀巢开发组合动作

## 背景与张力

雀巢说明了一个关键点：Exploit 开发组合管理不是静态维护，而是主动工作。成熟食品饮料集团可以拥有大型盈利品类，但仍然面对颠覆风险：健康需求变化、零售权力转移、高端咖啡增长、植物基食品出现，以及某些品类战略吸引力下降。

组合颗粒度不是 SKU，而是“品类 / 品牌平台”：饮料、咖啡、宠物护理、植物基食品、皮肤健康、糖果等。

左侧的植物基食品更接近 Explore：它测试新的消费趋势和价值主张。皮肤健康和糖果业务则提醒我们：一个业务仍可能盈利，但如果战略适配度下降，就要考虑 Improve、Divest 或 Dismantle。

## 动态移动

| 组合对象 | 起点 | 移动 | 组合动作 | 判断标准 |
|---|---|---|---|---|
| 核心饮料 / 咖啡 | Exploit 高收益区 | 强化和保护 | Improve / Acquire / Partner | 是否提升增长、品牌溢价和渠道控制 |
| 星巴克授权 | 相邻机会 | 转入开发组合 | Partner / Invest | 是否快速强化咖啡组合而非自建渠道 |
| Blue Bottle 等高端咖啡 | Explore / Transfer | 向高端增长区移动 | Acquire / Transfer | 是否带来新客群和高端能力 |
| 植物基食品 | Explore 选项 | 根据证据坚持或转向 | Invest / Pivot / Persevere | 健康趋势、复购、渠道接受度是否成立 |
| 皮肤健康 / 糖果 | Exploit 但适配度下降 | 修剪或退出 | Divest / Dismantle | 资本是否有更好用途，风险是否超过组合价值 |

## 机制

机制是有纪律的修剪和更新。雀巢不只是问“我们还能推出什么新产品？”它还问：哪个现有品类要强化，哪个要改善，哪个要出售，哪个相邻机会值得探索？这就是商业模式组合管理的核心。

## 风险与限制

组合动作也会隐藏执行风险。收购可能买贵，合作可能低于预期，剥离也可能过早移除未来选项。因此 Portfolio Map 必须随时间复盘，而不是一次性的投资者日图表。

## 迁移启发

对成熟公司来说，创新不只发生在 Explore 侧。Exploit 组合动作同样具有战略意义，因为它释放资本、降低颠覆风险，并让核心业务保持面向未来的适配度。
