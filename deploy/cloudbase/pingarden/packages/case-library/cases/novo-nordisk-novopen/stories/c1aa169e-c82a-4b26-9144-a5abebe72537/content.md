# 战略读法：Bain 价值元素

NovoPen 是很好的 **Bain 价值元素**案例，因为它改善的不只是注射功能。对糖尿病患者来说，任务包含剂量准确、隐私、信心、日常坚持，以及减少用药错误带来的恐惧。

功能价值包括质量、降低风险、节省时间和简化。情感价值包括降低焦虑和增强信心。在某些场景下，它甚至接近“改变生活”，因为更好的用药习惯会影响日常自由和掌控感。

VPC 读法不能把患者当成笼统用户。需要区分患者、照护者和医生的任务、痛点与收益。价值地图应具体说明设备设计、剂量记忆、便携性和可靠性如何缓解痛点，而不是只写“更好的设备”。

价值元素不能替代证据。医疗器械价值必须通过安全性、临床流程、依从性、报销和信任来验证。
