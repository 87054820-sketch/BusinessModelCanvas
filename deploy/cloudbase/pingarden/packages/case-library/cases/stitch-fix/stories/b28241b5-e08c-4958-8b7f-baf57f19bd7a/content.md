# 战略读法：Bain 价值元素

Stitch Fix 适合用 **Bain 价值元素**分析，因为它不是只在服装 SKU 上竞争，而是把功能价值和情感价值组合起来。顾客买的不只是衣服，而是把发现、筛选、自信和合身判断外包出去。

功能价值包括节省时间、减少精力、组织选择和提供多样性。情感价值包括降低焦虑、审美设计和建立风格身份。最重要的教学点是数据推荐和真人造型师判断的组合：技术缩小选择范围，真人层让结果显得个性化。

VPC 读法应从顾客痛点开始：选择过多、合身不确定、缺乏信心、不喜欢逛店。价值地图不能只写“个性化”，而要具体写清楚精选盒子、反馈循环、造型师说明、退货流程和学习数据。

价值元素解释的是客户吸引力，不是完整商业模式。案例仍需要 BMC 分析库存风险、退货、造型师成本、数据优势，以及个性化经济性是否能盈利扩张。
