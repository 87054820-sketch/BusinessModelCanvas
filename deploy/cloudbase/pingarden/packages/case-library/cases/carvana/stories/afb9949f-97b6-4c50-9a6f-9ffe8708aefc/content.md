# 战略读法：Porter 五力模型

Carvana 是 **Porter 五力模型**案例，因为线上二手车零售改变了一个分散、信任不足行业的压力结构。

行业内竞争来自经销商、交易市场、拍卖和线上零售商。买方力量很强，因为客户会比较库存、价格、金融方案、交付和退货政策。供应商力量体现在车源获取、整备能力、物流、金融伙伴和检测数据。替代品包括传统经销商、C2C 平台、租赁、网约车和推迟购车。新进入者可以做数字门店，但规模化物流和整备能力更难复制。

战略问题是 Carvana 的模式是否足以降低客户痛点，并抵消资本密集度。阅读 BMC 时要结合五力：透明价格和退货降低买方焦虑，而物流与整备既是优势来源，也是成本风险。

五力模型不能证明 Carvana 一定有吸引力。它说明这个商业模式必须同时赢在信任、库存周转、资本成本和运营规模上。
