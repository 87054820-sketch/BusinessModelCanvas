# 战略读法：Porter 五力模型

移动通信解绑是 **Porter 五力模型**案例，因为旧的一体化运营商模式同时受到多方向挤压。网络基础设施、手机平台、应用生态、OTT 通信、监管和客户，都拿走了运营商过去控制的一部分价值。

行业内竞争体现在覆盖、价格、套餐和降低流失率。供应商力量来自网络设备商、频谱资源、手机平台和应用商店。买方力量随着携号转网和价格透明而增强。替代品包括消息应用、VoIP、Wi-Fi 通话和平台内置服务。新进入者未必建设完整网络，但 MVNO 和数字品牌会攻击客户关系。

解绑是对 BMC 多处压力的战略回应。有的玩家专注基础设施规模，有的专注客户品牌，有的专注数字服务。五力模型解释了为什么当每项活动承受不同压力时，“什么都自己做”会变得不那么有吸引力。

五力模型不能替代详细经济性。监管、频谱拍卖、资本开支周期和国家市场结构都可能主导结果。先用五力定位压力，再用 BMC 重新设计活动系统。
