# Strategy lens: Porter Five Forces

Mobile telecom unbundling is a **Porter Five Forces** case because the old integrated operator model was squeezed from several directions at once. Network infrastructure, handset platforms, app ecosystems, over-the-top communication, regulators, and customers each captured part of the value that operators once controlled.

Rivalry is intense when operators compete on coverage, price, bundles, and churn reduction. Supplier power rises when network equipment vendors, spectrum owners, handset platforms, and app stores shape the operating system. Buyer power increases as number portability and transparent pricing make switching easier. Substitutes include messaging apps, VoIP, Wi-Fi calling, and platform-native services. New entrants may not build full networks, but MVNOs and digital-only brands attack customer relationships.

Unbundling is a strategic response to pressure across the BMC. Some players specialize in infrastructure scale, some in customer brands, and some in digital services. Five Forces helps explain why “own everything” becomes less attractive when each activity has a different pressure structure.

The framework is not a substitute for detailed economics. Regulation, spectrum auctions, capex cycles, and national market structure can dominate the story. Use Five Forces to locate pressure, then use BMC to redesign the activity system.
