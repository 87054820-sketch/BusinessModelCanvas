# 战略读法：McKinsey 7S 与 Porter 五力模型

Patagonia 同时标记 **McKinsey 7S** 和 **Porter 五力模型**，因为它既是组织一致性案例，也是行业压力案例。

7S 读法从 Shared Values 开始：环保行动不是商业模式之后追加的营销活动。它影响 Strategy、产品选择、供应链 Systems、招聘、沟通 Style，以及围绕维修、材料和倡议建立的 Skills。这个案例成立，是因为软要素和硬要素互相强化。

五力读法解释这种一致性为什么有竞争意义。户外服装行业面对同业竞争、负责任材料的供应压力、买方审视、低价服装替代品，以及可以复制可持续话术的新进入者。Patagonia 不是逃离这些力量，而是改变竞争基础：信任、维修、耐用和使命降低了单纯价格比较。

迁移启发是纪律。使命主张只有在 7S 系统支撑它，并且改变企业应对行业力量的方式时，才真正成为战略；否则只是营销语言。
