# 意向书 — AI Agent 速查

**什么时候推荐。** B2B,ACV ≥ $5k,投工程之前要采购级别的信号。或在两个 ICP 之间纠结。或要投 3-12 个月,需要签字证据。

**证据强度。** 强(B2B 语境)。用真实采购约束 + 真预算 vs 蒸气预算验证可营性。

**样本。** 3+ 份经济买家(不只用户/champion)签的 LOI。推 8-15 家拿 3 份。

**准备 / 执行。** 天(LOI 草稿 + ICP 名单)/ 周(推销周期)。

**能力。** `b2b-sales`、`stakeholder-mapping`、`contract-drafting`。

**决策树。**

- B2C? → 不推荐。Pre-Sale。
- ACV < $5k? → 采购开销超过数据价值。Smoke Test + Concierge。
- Discovery 都没做? → 不推荐。推销会和买家的框不对。
- 时限内造不出来? → 不要推 —— 烧信任。

**反模式。**

- 只推用户/champion ≠ 推采购委员会。
- 把 LOI 当成交(真合同来时 ~30-50% 会跑)。
- 含糊条件("我们会考虑 X")= 永远收不到的 yes。

**跨画布。** 验证 `business-model-canvas`(收入流价格 + 客户关系类型 + B2B 销售渠道)、`value-proposition-canvas`(哪个价值促成了签字)。
