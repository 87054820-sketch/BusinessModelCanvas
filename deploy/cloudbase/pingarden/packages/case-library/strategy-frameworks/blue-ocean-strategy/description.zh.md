# 蓝海战略

> 蓝海战略不是一种商业模式类型，而是一套战略分析框架：通过重构市场边界，设计一条让直接竞争变得不那么重要的新价值曲线。

## 为什么这个方法重要

蓝海战略要求团队不要把当前行业规则视为固定不变。它不是让企业在同一组维度上比竞争对手做得更高，而是寻找一条新的价值曲线：重点突出、形状不同，同时创造新需求，并降低或移除客户不再重视的成本项。蓝海战略官网的定义是：同时追求差异化和低成本，以开创新市场空间并创造新需求（BOS official site）。

对 PinGarden 来说，这套方法特别适合连接“战略分析”和“商业模式设计”。战略画布负责看清当前竞争曲线；ERRC 四步动作解释战略取舍；商业模式画布则进一步检验这条新价值曲线背后是否有完整的运营模型。《商业模式新生代》（BMG 2010）也明确指出：价值侧的改变必须回看资源、活动、伙伴和成本结构的连锁影响。

## 核心工具

1. **战略画布**：把竞争者放在同一组竞争因素上比较。目标不是所有因素都更高，而是形成明显不同的曲线形状。
2. **ERRC 四步动作**：思考剔除、减少、提升、创造。剔除和减少节省的成本，应该用来支持提升和创造。
3. **非顾客**：不只研究现有客户，还要看拒绝、回避或从未考虑过当前行业供给的人。
4. **市场边界重构**：挑战行业对买方群体、功能/情感导向、替代行业、互补产品和时间趋势的默认假设。

## 一个好的蓝海案例应该包含什么

一个好的案例不能只停留在“创新”口号上，而应该说明：

- 原行业的红海竞争逻辑；
- 5–10 个客户真实比较的竞争因素；
- 至少两条价值曲线：行业常规曲线和蓝海行动曲线；
- 清晰的 ERRC 解释；
- 被打开的非顾客或未充分服务买方；
- 对商业模式画布的影响：渠道、活动、资源、伙伴、收入和成本如何变化。

## PinGarden 第一批案例

- `cirque-du-soleil`：剔除动物和明星演员，提升剧场式艺术体验，创造面向成人和企业客户的高端现场娱乐。
- `yellow-tail`：降低葡萄酒选择和品鉴门槛，把啤酒、鸡尾酒等大众酒精饮品消费者拉入葡萄酒市场。
- `qb-house`：把理发店压缩成 10 分钟功能型快剪，并围绕速度和确定性重构运营。
- `nintendo-wii`：避开主机硬件性能军备竞赛，为休闲玩家和家庭用户创造体感游戏体验。

## 常见错误

- **处处比对手高的幻想。** 如果一条曲线在所有因素上都高于竞争对手，通常意味着成本爆炸，而不是战略。
- **竞争者评分没有证据。** 战略画布的分数应来自客户评论、公开价格、功能对比或可信案例资料。
- **先做 ERRC 再画曲线。** ERRC 应该由价值曲线对比推导出来，而不是脱离市场的头脑风暴。
- **只有创新，没有商业模式闭环。** 新价值曲线不够，BMC 还必须说明企业如何交付价值并捕获价值。
