# McKinsey 7-S Framework

## Select this framework when

- The user has a strategy but execution is blocked by organization design, systems, skills, staff, leadership style, or values.
- The case involves transformation, integration, scaling innovation, or operating-model change.

## Use these canvases

1. `innovation-culture-map` — map behaviors, enablers, blockers, and outcomes.
2. `design-criteria-canvas` — turn alignment needs into design rules.
3. `business-model-canvas` — check what the organization must actually deliver.

## Questions to ask

- Which strategic choice must the organization execute?
- Which 7-S elements reinforce it?
- Which elements contradict it?
- What behavior or mechanism must change first?

## Red flags

Do not list seven categories without diagnosing interactions. Do not treat values as slogans. Do not ignore systems and incentives.
