# 搜索趋势分析 — AI Agent 速查

**什么时候推荐。** 访谈出了具体的问题语言,用户想看它在公开网络上有没有规模。或两个候选细分相对定大小。或对 BMC Environment 的"关键趋势"做现实检验。

**证据强度。** 弱。确认词汇走得远 + 趋势方向;**不确认** 付费意愿或以美元计的可触达市场。

**准备 / 执行。** 小时 / 小时。

**能力。** `keyword-research`、`trend-analysis`。

**决策树。**

- 细分是分析师驱动采购的 B2B? → 不推荐。搜索信号弱。改 Customer Interview + 行业特定场子的 Discussion Forums。
- 5 年趋势线下行? → 把这一点作为红旗点出来 —— 逆趋势进入市场是已知的 pitfall。
- Google 量高但 Reddit / HN 不讨论? → 大概率被 SEO 吃掉了。给数字加 caveat。
- 用户要 $ 计的市场规模? → **不要** 只推搜索趋势。配 Letter of Intent / Pre-Sale 看真实需求。

**反模式。**

- 一个头部词 ≠ 全貌。要 5-15 个。
- 30 天小高峰 ≠ 趋势。要看 5 年窗口。
- 量级 ≠ TAM。

**跨画布。** 验证 `business-model-environment` 的关键趋势那一块、按相对量级锐化 `value-proposition-canvas` 痛点。
