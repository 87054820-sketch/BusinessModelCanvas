# Strategy Framework Case Review

This review captures the current quality debt for strategy-framework tags. A tag is not complete unless story and/or canvas content explains how the framework applies.

## Review rules

- Keep a framework tag only when a reader can understand the framework logic from story and canvas content.
- Add a framework-specific story when the original business-model story should be preserved.
- Add or improve a canvas when the framework is visual by nature.
- Remove weak tags when the case is only generally innovative.

## Blue Ocean Strategy

Blue Ocean stories must explain red-ocean baseline, noncustomers, ERRC, value curve, and BMC consequences. Cases that only have a Strategy Canvas or BMC without explanatory story need story upgrades.
