## Method contribution

Compared with Peter Schwartz’s *The Art of the Long View*, this book contributes an organizational execution layer:

- project purpose and sponsor support before scenario work begins;
- structured exploration before building scenarios;
- scenario development as a disciplined, facilitated process;
- implementation through strategy discussion, wind-tunneling, mental-model change, and action design;
- project assessment and documentation of results;
- attention to stakeholder participation, perception, and organizational learning.
